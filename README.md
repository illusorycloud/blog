环境配置步骤,写给自己，备忘。

* 1）clone 此仓库

```shell
git clone git@github.com:lixd/blog.git
```

* 2）创建`themes`文件夹并克隆自己喜欢的题主

```shell
mkdir themes
cd themes
# 克隆到maupassant目录下
git clone git@github.com:lixd/maupassant-hugo.git maupassant
```



hugo 常用命令

```shell
# 查看hugo版本号
hugo version 
# 本地运行
hugo server
# 生成public文件
hugo
```
